📡 API Endpoints

# Admin APIs
GET    /api/v1/admin/users             # List users
POST   /api/v1/admin/users             # Create user
GET    /api/v1/admin/users/{id}        # Get user detail
PATCH  /api/v1/admin/users/{id}        # Update user
DELETE /api/v1/admin/users/{id}        # Delete user

# Provider APIs
GET    /api/v1/providers/schemas       # Get all schemas
POST   /api/v1/providers/test          # Test provider

# TTS Direct APIs
POST   http://localhost:8101/synthesize  # Valtec TTS
POST   http://localhost:8102/synthesize  # VieNeu TTS

🚀 Quick Start

1. Test TTS Services

# Valtec TTS
curl -X POST "http://localhost:8101/synthesize" \
  -H "Content-Type: application/json" \
  -d '{"text": "Xin chào", "speaker": "female"}'

# VieNeu TTS
curl -X POST "http://localhost:8102/synthesize" \
  -H "Content-Type: application/json" \
  -d '{"text": "Xin chào", "speaker": "female"}'

2. Run Feature Tests

bash scripts/test_features.sh
python3 scripts/test_admin_api.py
